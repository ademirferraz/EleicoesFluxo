Como utilizar este painel:
Inicie seu servidor: Rode o comando python app.py no terminal.

Abra o arquivo: Basta dar um duplo clique no arquivo admin.html (ou arraste-o para o navegador).

Atenção ao CORS: Se o navegador bloquear o envio por segurança, você precisará instalar a biblioteca flask-cors no seu Python:

Rode: pip install flask-cors

No app.py, adicione:

Python

from flask_cors import CORS
app = Flask(__name__)
CORS(app) # Isso libera o acesso para o seu HTML<\PRE>
